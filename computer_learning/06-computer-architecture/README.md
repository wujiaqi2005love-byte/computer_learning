## 推荐学习方式

1. **先读 README** — 理解概念和原理
2. **跑 notebook** — 每个 notebook 都包含可交互的 Python 模拟代码，`Shift + Enter` 逐个 cell 运行
3. **做 ✏️ 练习** — 每个 notebook 有互动编程题
4. **完成 🎯 综合挑战** — 每章末尾的练习题

> 💡 这个系列不要求你有硬件背景。只要会 Python，就能通过模拟器理解计算机内部是如何工作的。
